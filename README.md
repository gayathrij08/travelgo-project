# ✈️ TravelGo - Smart Travel Booking Web App

TravelGo is a simple and user-friendly web application that allows users to search and book flights, hotels, and manage their bookings easily.

---

## 🚀 Features

- 🔐 User Authentication (Login / Signup / Forgot Password)
- 🏠 Dashboard with personalized welcome
- ✈️ Flight Search & Booking
- 🏨 Hotel Search
- 📄 My Bookings Page
- 💳 Payment Page
- 🎨 Attractive UI with responsive design
- 💾 LocalStorage used for storing user and booking data

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage (for frontend data handling)
